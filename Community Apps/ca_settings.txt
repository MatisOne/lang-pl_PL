Hide Deprecated Applications=Ukryj Wycofane Aplikacje
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections= To ustawienie ukryje wycofane aplikacje z listy dostępnych aplikacji Wycofane aplikacje będą jednak nadal wyświetlane w sekcji zainstalowanych i poprzednich aplikacji 
Hide Incompatible Applications=Ukryj Niekompatybilne Aplikacje
This setting will hide applications that are listed as being incompatible with your version of Unraid=To ustawienie ukryje aplikacje, które są wymienione jako niezgodne z Twoją wersją Unraid
Enable Reinstall Default=Włącz Domyślne Reinstall
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Ta opcja uaktywnia przycisk na kartach i wyskakujących okienkach, co pozwala na ponowną instalację zainstalowanej aplikacji przy użyciu domyślnych wartości Ta opcja może być myląca i jest używana tylko wtedy, gdy chcesz zainstalować i mieć uruchomione aplikacje równolegle
Enable additional search results from dockerHub=Włącz dodatkowe wyniki wyszukiwania z dockerHub
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=To ustawienie umożliwia CA pobieranie dodatkowych wyników wyszukiwania z dockerHub Należy pamiętać, że docker musi być włączony, aby to działało
Allow CA to send any emergency notifications=Zezwól CA na wysyłanie powiadomień awaryjnych
This setting allows CA to periodically scan your system and alert you to the presence of=To ustawienie umożliwia CA na okresowe skanowanie systemu i ostrzeganie o obecności plików
